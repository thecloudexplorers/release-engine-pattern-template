# Documentation Downloads

This directory contains pre-generated PDF and DOCX versions of all Release Engine Workload Patterns documentation.

## 📚 Available Documents

### Main Documentation
- **README.pdf** / **README.docx** - Complete repository overview
- **PATTERNS_OVERVIEW.pdf** / **PATTERNS_OVERVIEW.docx** - Comprehensive patterns guide

### Pattern-Specific Documentation
- **resource_group_scope_pattern.pdf** / **resource_group_scope_pattern.docx** - Basic pattern
- **subscription_scope_pattern.pdf** / **subscription_scope_pattern.docx** - Intermediate pattern
- **multi_stage_pattern.pdf** / **multi_stage_pattern.docx** - Advanced pattern

## 🔄 Automatic Generation

These files are automatically regenerated by the GitHub Actions workflow whenever:
- Markdown documentation is updated in the repository
- Changes are pushed to the main branch
- The generation workflow is manually triggered

## 🛠️ Manual Generation

To regenerate these files locally, run:

```bash
bash scripts/generate-docs.sh
```

This requires Pandoc and wkhtmltopdf to be installed on your system.

## 📝 Format Information

- **PDF files**: Best for printing, archiving, and reading. Optimized for A4 paper size.
- **DOCX files**: Editable Microsoft Word format, ideal for customization and collaboration.

**Note**: Mermaid diagrams in the source markdown are not rendered in the exported formats. For diagrams, please refer to the online documentation or the GitHub repository.
